Aliapur's Starter Kit

Aliapur has developped a Beta version of its Starter Kit to prototype objects by moulding them. Also you can create coverings. It is now available for free (France only).
The aim is to provide the curious makers with samples of raw recycled materials, so that can create their first objects and imagine new possibilities !

  • Rubber powder & granulates - from 0 to 0.8mm, from 1 to 3mm and from 3 to 6mm (each 1kg)
  • A polyurethane binder (1 kg can)

2 minutes to discover the process

Why this platform ?

This plaform is based on Github, with

  • a repository to gather together the realisations made by YOU with the starter kit
  • a WIKI with informations about the materials, the process or the existing reuses

Objectives

  • To be a state of the art for the recycling of used tyres
  • To inspire new products and uses that would give used tyres a second life
  • To be open source

How to contribute ?

  • You can IMPROVE the WIKI with advices about the process, the reuses, the technical precision for the materials
  • YOU can CREATE a file in MarkDown (.md) with the documentation of your creations and propose it to the repository by a pull request. To create Ã markdown file you can use framapad.org or hackmd.io for example.

What is Aliapur ?

Aliapur is the leader company in the field of recovering end-of-life tyres in France. It is a Public Limited Company whose founding members are Bridgestone, Continental, Dunlop Goodyear, Kléber, Michelin and Pirelli. (SOURCE:aliapur.fr/en)
